## Usage

### Install

```sh
npm install
```

### Dev

Requirements: jsbox-cli, code sync and build .box file
usage: https://github.com/Dreamacro/jsbox-cli#readme

```sh
npm run dev && npx jsbox watch dist
```

### Build

```sh
npm run build
```
